SM_user_guide_v2_superseded.pdf: 
--------------------------------
User guide for original version of the dataset,
before the paper was reviewed. This version was using SMOS data, which was later
replaced by ASCAT. There were some errors in the TCA algorithm. 
The TCA soil moisture product was produced at 9km resolution,
whereas in the final version, it is at 12.5km resolution.

SM_user_guide_v3.pdf:
---------------------------------
User guide for final version of the dataset (after paper was reviewed and
methodology was revised). Errors in TCA algorithm have been corrected.
ASCAT is used instead of SMOS.
Spatial resolution of the final product is 12.5km.
